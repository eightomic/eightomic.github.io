About

I'm William Stafford Parsons, a computer science researcher, game developer and web software engineer with an emphasis on minimalism.

I create tiny offline browser games that are fun to play for a few minutes at a time with smooth gameplay.

I conduct open computer science research to create improved fundamental algorithms and data structures.

I'm starting BitVanish, a collection of offline browser tools for compressing and converting formatted data with client-side privacy.

Furthermore, I aspire to contribute impactful enhancements to useful projects in the open-source software industry as a betterment to society.
