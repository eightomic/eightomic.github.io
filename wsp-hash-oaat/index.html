WSP-Hash-OAAT: A Fast, Lightweight, Non-Multiplicative, One-at-a-Time 32-Bit Hashing Algorithm That Passes Rigorous Collision Tests

I developed a tiny 32-bit OAAT hashing algorithm as a practical alternative to FNV-1a, Good OAAT, Jenkins OAAT, Micro OAAT and Murmur OAAT, but I think it was too similar to Good OAAT without enough unique benefits.

The following new hashing algorithm as a lossless improvement to 32-bit FNV-1a and Micro OAAT is in development instead with a full library, reference and explanation comping soon.

#include <stdint.h> uint32_t wsp_hash_oaat(unsigned long input_count, const uint8_t *input) { uint32_t _state = 1; uint32_t state = 1111111111; unsigned long i = 0; while (i != input_count) { state ^= input[i]; state += state << 3; _state += state; _state = (_state << 27) | (_state >> 5); i++; } state ^= _state; state = (state ^ _state) + ((state << 10) | (state >> 22)); state += (_state << 27) | (_state >> 5); return state; }
